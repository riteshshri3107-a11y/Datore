# Datore — Trusted Skills & Social Marketplace
By **AARNAIT AI** | [datore.vercel.app](https://datore.vercel.app)

## Features
- 🛡️ Police verification badges & QR safety scan
- 💰 Escrow payments (99% worker / 1% platform)
- 🏪 Full marketplace (18 categories: Vehicles, Electronics, etc.)
- 👥 Community groups & social networking
- 📝 Create posts, stories, reels, events, fundraisers
- 💾 Saved items, memories, inbox
- 🌗 Dark/Light theme toggle
- 🤖 AI chatbot assistant
- 💬 Real-time chat
- 👑 Admin dashboard
- 📱 Mobile-first, PWA-ready

## Tech Stack
Next.js 14, Supabase, Tailwind CSS, Zustand

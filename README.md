# Datore - Peer-to-Peer Skills Marketplace
By **AARNAIT AI** | [datore.vercel.app](https://datore.vercel.app)

## Tech Stack
Next.js 14, Supabase (Auth + PostgreSQL + Realtime), Tailwind CSS, Zustand

## Features
- Police verification badges
- QR safety profiles
- Escrow payments (99% worker / 1% platform)
- Dual rating system
- AI chatbot
- Real-time chat
- Admin dashboard
- 12+ skill categories

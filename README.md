# 🛡️ Datore v5.0 — Safe Service Marketplace + JobPlace

## Features
- 💼 **JobPlace** — Post jobs, find workers, real-time map view
- 🗺️ **Live Map** — See available workers with ratings, prices, trust scores
- 🛡️ **QR Verification** — Scan worker QR codes to verify identity
- 🤖 **AI Trust Score** — Behavioral trust rating (0-100)
- 🏪 **Marketplace** — Buy & sell across 18 categories
- 💰 **Token Wallet** — Escrow payments with tip controls
- 👥 **Community** — Skill groups and local networks
- 🌗 **Glassmorphism UI** — Dark/Light theme with customizable glass effects
- 🎨 **Accent Colors** — 8 customizable accent colors
- 📱 **Mobile-first** — Responsive design, PWA-ready

## Tech Stack
Next.js 14, Supabase, Tailwind CSS, Leaflet Maps, Zustand

## Setup
```bash
npm install
# Set env vars in .env.local
npm run dev
```

## Environment Variables
```
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
```
